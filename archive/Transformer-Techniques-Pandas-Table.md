| # | Category | Pandas Functions | Description |

| 1 | Filtering & Selection | `df.query()`, `df.dropna()`, `df.loc[]` | Filter data by status, conditions, non-null values |
| 2 | Merging & Joining | `df.merge()` | Left join, inner join, multiple keys |
| 3 | Mapping & Lookups | `df.map()`, `df.apply()` | Dictionary mapping, dataframe lookups |

| 5 | Date Handling | `pd.to_datetime()`, `df.dt.date` | Parse, format, handle invalid dates |
| 6 | Aggregation & Grouping | `df.groupby()`, `df.agg()` | Group and aggregate, concatenate strings |
| 7 | Type Conversion | `df.astype()`, `pd.to_numeric()` | Convert to numeric, integer, boolean |
| 8 | Conditional Logic | `np.where()`, `df.apply()` | Conditional with numpy, lambda apply |

| 12 | Complex Transformations | Multiple chained operations | Multi-step transformations |
